== Description ==

Stratis is a plugin that adds a shortcode with a form to create a new unpublished post and send an email to the administrator.

== Installation ==

    Upload the entire stratis folder to the /wp-content/plugins/ directory.
    Activate the plugin through the 'Plugins' menu in WordPress.
    Use the shortcode [stratis_post_formulaire] on a page or post to display the form.

== Frequently Asked Questions ==

= How can I customize the form fields? =

You can modify the form fields by editing the stratis.php file in the plugin directory.
